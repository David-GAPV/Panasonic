# Quick Start

> This document is superseded by [README.md](README.md).

## Access

1. Open https://idp.pngha.io.vn in your browser
2. Log in with your assigned credentials (admin, reviewer, or uploader role)
3. Dashboard shows document statistics and recent uploads

## Basic Workflow

1. **Upload** — Go to Upload page, select files (PDF/DOCX/images), enter a Shipment Reference, submit
2. **Review** — Flagged documents appear in the Review Queue. Open to see side-by-side view of original document and extracted fields
3. **Edit** — Click any field value to correct it. Validation re-runs automatically after each edit
4. **Approve/Reject** — Use the action buttons. SAP payload is generated on approval. SNS email notification sent

## Roles

- **admin** — Full access including SAP simulation
- **reviewer** — Can review, edit, approve/reject, view analytics and reports
- **uploader** — Can upload and view documents (read-only detail, no review actions)

See [README.md](README.md) for full feature list and architecture.
